# Verb Bank for Resume Bullets

Use this bank to ensure no verb is repeated within the same experience entry.
Pick verbs that echo language used in the target JD when possible.

## Usage Rules

1. Within one experience entry, each bullet must use a **different verb**
2. Across the whole resume, try to use each verb **at most 2 times**
3. Always match verb **tense**: past tense for previous roles, present tense for current role
4. Prefer verbs that appear in the **JD's responsibilities section** (ATS signal)
